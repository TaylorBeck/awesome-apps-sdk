## üß∞ What is the Apps SDK?

The **Apps SDK** is an open-source toolkit by OpenAI for building custom apps that integrate directly into ChatGPT. It lets you define rich user interfaces, connect external tools or APIs, and combine them with the [Model Context Protocol (MCP)](https://github.com/modelcontextprotocol) ‚Äî all without leaving the chat interface.

* üßë‚Äçüíª Build **interactive apps** inside ChatGPT
* üîå Combine with MCP servers for deeper integrations
* üß™ Works with Python, TypeScript, and more
* üì¶ Fully open source and extensible
